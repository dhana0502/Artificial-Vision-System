# Artificial Vision System for Visually Impaired

An AI-based vision assistance system developed to help visually impaired individuals using Computer Vision and Deep Learning technologies.

## Features

- Real-time face detection
- Object recognition using OpenCV
- Emotion, gender, and race prediction
- Audio feedback using text-to-speech
- User-friendly assistance system

## Technologies Used

- Python
- OpenCV
- Flask
- TensorFlow
- DeepFace
- MySQL

## Project Description

This project helps visually impaired users understand their surroundings through real-time image processing and audio guidance. The system captures visual data, processes it using AI models, and provides voice-based feedback to assist users.
